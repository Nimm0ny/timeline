# 编年 Chronicle · 移动端 Web 形态 — 设计基准（已批准并落地）

> 本文是「移动端 web 形态」的实现基准。**仅做响应式 web 移动形态，不做原生应用。** 主打时间线笔记，风格延续 Obsidian/Notion 极简。
> 状态：**已由用户批准并落地**。移动端经本文定义的独立断点基准实现；桌面 `>1024px` 仍以 Obsidian Minimal 三栏基准为准。
> 已同步：spec §0/§9/§12 与 `AGENTS.md §9` 将「禁止移动端重排」收敛为「禁止未按本文基准擅自移动端重排」。

## 0. 目标 / 非目标
- **目标**：桌面三栏在窄屏（手机/竖屏）下自适应为可用的单栏移动形态；触控友好；复用既有令牌/组件/主题（含暗色）。
- **非目标**：原生 App、离线 PWA 壳、手势库、移动专属新功能。先把「看时间线 + 读/记笔记 + 组织属性」这条主闭环在手机上跑通。

## 1. 断点与总策略
- 断点：`≤ 768px` = 移动模式（`mobile`）；`769–1024px` = 紧凑桌面（沿用桌面三栏，左右栏更窄）；`> 1024px` = 现状桌面基准（不变）。
- 实现：CSS media query 为主 + 一个 `useViewport()` 复合式提供 `isMobile` 给 `TimelinePage` 做**单栏路由**（决定当前显示哪一栏）。**桌面 `> 1024px` 行为零改动**（不破坏现有基准/视觉 QA）。
- 不变量延续：无 `transform:scale` 承载布局、全局禁滚动条、右栏阅读↔编辑零位移无边框、令牌纪律、禁装饰渐变/glow/玻璃态。

## 2. 移动布局（单栏栈 + 抽屉，参考 Obsidian/Notion 手机版）
桌面三栏 → 移动「一次只显示一栏 + 导航」：
```
┌─────────────────────────┐
│ ☰  党史 · 共56条     🔍 ＋│  顶栏：抽屉/标题/搜索/新建
├─────────────────────────┤
│  时间线列表（中栏为主屏） │  纵向列表；行点按 → 详情全屏滑入
│  ...                      │
│                           │
└─────────────────────────┘
   ▲ 左滑出抽屉：笔记本/视图/属性/统计/设置
   ▲ 点行 → 详情全屏（左上角 ← 返回）
```
- **主屏 = 中栏**（时间线列表）。左栏（笔记本/视图/属性/统计 + 设置/帮助）收进**左侧抽屉**（顶栏 ☰ 打开，点遮罩/选项后关闭）。
- **详情/编辑 = 全屏**：点某条 → `EventDetailPane` 全屏滑入，左上角 `←` 返回列表（复用现有 rightOpen 状态，移动下渲染为全屏层而非右栏）。
- **设置**：已是全屏 modal，移动下天然可用（导航改为顶部横向或可折叠）。
- 三栏拖拽 resizer 在移动下隐藏；hover-only 的星标/操作改为**常显**（触控无 hover）。

## 3. 触控与输入
- 命中区 ≥ 44px：行高、图标按钮在移动下放大到触控规格（新增移动专用尺寸令牌或 media 覆盖既有 `--icon-*`/行高，仅 mobile 生效）。
- 弹层（OptionPicker/列设置/时间定位）：移动下改为**底部抽屉式 sheet** 或全宽弹层，避免窄屏溢出裁剪（解决桌面下中栏行内弹层被滚动容器裁剪的同类问题）。
- 无感编辑器：`contenteditable` 触控可用；移动下工具条精简；图片插入走系统文件选择。

## 4. 组件复用与改动映射
- 复用：`TopicSidebar`（抽屉容器内）、`TimelineFeed`（主屏）、`EventDetailPane`（全屏层）、`SettingsModal`、主题系统（令牌自适应，暗色直接可用）。
- 改动：`pages/TimelinePage.vue`（`isMobile` 单栏路由 + 抽屉开合状态 + 全屏详情层）、`styles/timeline-notes.css`（`@media (max-width:768px)` 布局/尺寸覆盖 + 抽屉/顶栏样式）、新增 `composables/useViewport.js`、可能新增 `components/MobileTopBar.vue`。
- 数据契约/后端：**零改动**（纯前端响应式）。

## 5. 分阶段（建议）
- **M1 响应式外壳**：断点 + `useViewport` + 顶栏 + 左抽屉 + 单栏路由（列表↔全屏详情）+ resizer 隐藏 + 桌面零回归。
- **M2 触控打磨**：命中区/行高/图标移动尺寸；弹层改底部 sheet；星标/操作常显。
- **M3 收尾**：移动空状态、设置导航折叠、移动视觉 QA（375×812 / 390×844 归档）。

## 6. 已落地关键决策
1. 主屏默认是**中栏时间线列表**，笔记本/视图/属性/统计进入左抽屉。
2. 左栏导航采用**左侧滑出抽屉**（Obsidian 风），不引入底部 Tab 栏。
3. 断点采用 `768px`；`769–1024px` 为紧凑桌面三栏，`>1024px` 沿用桌面三栏基准。
4. 弹层在移动端统一改为**底部 sheet** 或全宽弹层，避免窄屏溢出裁剪。
